Final Project Submission Instructions
Hello Class,

As we approach the end of the semester, please find below the submission instructions for your final project.

Directory Structure

Please organize all project files — including source code, datasets (or links to datasets), notebooks, reports, and any supporting documents — into a single, clearly named directory. Use the following naming convention for your directory:

FinalProject_[YourLastName]_[YourFirstName]

For example: FinalProject_Smith_John

Your directory should include, at minimum, the following:

All source code and/or RMarkdown/Jupyter notebooks
A README file explaining the contents of the directory, how to run your code, and any dependencies or environment requirements
Your final project report (PDF format preferred)
Any additional resources referenced in your project (configuration files, result outputs, figures, etc.)
Sharing Instructions

Once your directory is ready, share it with me at gudivadav15@ecu.edu and grant read access. Do not modify any files after the submission deadline, as I will review the directory as it exists at that time.

Submission Deadline
