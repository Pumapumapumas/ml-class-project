# Archive

Older versions of guide content kept for historical reference. Files here
are not part of any current workflow. The active onboarding doc is at
[`../teammate_onboarding.md`](../teammate_onboarding.md).

## Files

- `teammate_onboarding_windows_native.md` — the pre-VM-pivot onboarding
  doc that walked a Windows teammate through installing WSL2, Docker
  Desktop, VS Code, the WSL extension, the GitHub CLI, and the project's
  Python/Docker toolchain on their own laptop. Superseded on
  2026-06-23 when we pivoted to giving the teammate SSH access to a
  pre-provisioned project VM, which collapsed most of that setup work.
