# Development planning

Implementation planning and tracking for the OCR project. This directory holds:

- **`roadmap.md`** — the high-level view: phases, milestones, current status
- **`phase_<N>_<name>.md`** — per-phase implementation plans with task checkboxes
- **`loose_ends.md`** — items noted during development that we deferred; consulted before each phase boundary
- **`reviews/`** — review notes, CPI cycle outputs, sprint retrospectives (created as needed)

Conventions and structure are defined in [`docs/standards/documentation_standard.md`](../standards/documentation_standard.md).
